# Project Overview

This project is a **static portfolio website** created using only HTML and CSS.  
It is designed as a simple one-page layout that introduces a developer, showcases skills, displays sample projects, and provides basic contact information.

The main goals of the project were:
- To practice building a responsive webpage layout
- To use Flexbox and CSS Grid for modern layout design
- To apply media queries for mobile responsiveness
- To demonstrate clean folder structure and documentation using Git & GitHub

The website includes the following sections:
1. **Header & Navigation**
2. **Hero Section**
3. **About**
4. **Skills**
5. **Projects**
6. **Contact**
7. **Footer**

No JavaScript frameworks or advanced tools were used, keeping the project simple and beginner-friendly while still showcasing modern web design concepts.
